CRM для ПВЗ-сети

Архитектура системы

Путь данных: от ПВЗ до тепловой карты
Слой 1 · Сбор
API маркетплейса → CRM каждого ПВЗ
Заказы приходят на точку: SKU, количество, время. CRM (TotalCRM, WireCRM) автоматически стягивает данные через API WB/Ozon каждые 30 мин.
Инструменты: ApiMonster, SelSup, Albato
Слой 2 · Обезличивание
Персональные данные → Хэш или удаление
ФИО, телефон, адрес заменяются на анонимный ID. Товар, количество, дата, геоячейка (500×500м) — сохраняются. Это происходит до передачи в центральное хранилище.
Требование: 152-ФЗ, необратимость
Слой 3 · Хранение
Data Warehouse (ClickHouse / PostgreSQL)
Обезличенные записи со всех тысяч ПВЗ агрегируются в единую базу. Схема: [геоячейка, категория, SKU, кол-во, дата, маркетплейс].
Объём: ~1 ТБ/год при 5000 ПВЗ
Слой 4 · Аналитика
BI + ML-модели прогнозирования
DataLens или Power BI строят отчёты: топ товаров по районам, сезонность, рост/падение категорий. ML предсказывает будущий спрос.
Инструменты: DataLens, Power BI, Python ML
Слой 5 · Геосервис
Тепловые карты + API для клиентов
Клиенты (бизнес, селлеры) видят интерактивные карты спроса по районам. API позволяет интегрировать данные в их системы. Геоинтеллект — для визуализации слоёв.

Как работает тепловая карта в реальном геосервисе
Каждая ячейка (500×500м) получает агрегированный показатель из обезличенных данных ПВЗ: количество заказов категории X за последние 30 дней. Интенсивность цвета = относительный спрос. Бизнес видит: «В квадрате A3 за месяц выдано 847 единиц электроники» — и принимает решение об открытии точки.

Дорожная карта

От идеи до масштаба
Месяц 1–2 · MVP
Сбор данных с 5–10 ПВЗ
Договориться с партнёрами, подключить TotalCRM + API WB/Ozon, собрать первые обезличенные данные. Проверить гипотезу: есть ли спрос на геоаналитику у бизнеса?
Результат: первый датасет + 1–2 платящих клиента
Месяц 3–6 · Продукт
Масштаб до 50–100 ПВЗ + DataLens дашборд
Разработать стандартный отчёт по локации. Запустить первые продажи геоотчётов (10–30 тыс. ₽). Подключить геовизуализацию (Геоинтеллект или собственный вариант).
Результат: 10+ клиентов, MRR 100–300 тыс. ₽
Месяц 7–12 · Масштаб
500+ ПВЗ, API для клиентов
Запустить подписочную модель API-доступа. Привлечь первых якорных клиентов — ритейл-сети и крупных селлеров. Расширить покрытие на 3–5 городов.
Результат: MRR 1–3 млн ₽, B2B-контракты
Год 2+ · Монополия данных
1000+ ПВЗ, эксклюзивный актив
С ростом числа точек данные становятся уникальным активом, который невозможно воспроизвести быстро. Переход к ML-прогнозированию спроса и SaaS-модели.
Результат: продажа данных крупным игрокам или привлечение инвестиций

Россия имеет уникальное преимущество: закон обязывает все кассы передавать данные онлайн через ОФД. Это создаёт единый поток чековых данных с детализацией до SKU — чего нет ни в одной другой крупной стране. На Западе аналогичный уровень детализации достигается только через дорогостоящие панели и партнёрства с ритейлерами.
D2C vs Маркетплейс

4 стратегии победы

1. Стать «маркетплейсом внутри себя»
Причина: покупатель идёт на Ozon за выбором → Решение: D2C-сайт предлагает экосистему (кофеварка + зёрна + фильтры + кружки) → Следствие: экономия 30% на комплекте + удобство одной корзины.
2. Инвертировать логистику
Причина: Маркетплейс = центральный склад → долго в регионы → Решение: собственные ПВЗ / партнёрская выдача / самовывоз со скидкой → Следствие: неудобство превращается в осознанный выбор рационального покупателя.
3. Выиграть на возвратах и гарантии
Причина: Маркетплейс → 30% возвратов в одежде → Решение: продлённая гарантия, бесплатный ремонт, обмен с доставкой → Следствие: доверие выше при той же или меньшей цене.
4. Подписка как крючок
Причина: Корм для питомца — регулярная покупка → Решение: «каждые 2 недели со скидкой 15%» → Следствие: удобство выше, чем на маркетплейсе, цена ниже, LTV максимальный.
Примечание
Подписка работает только для регулярных покупок при стабильном доходе аудитории. Для ценочувствительной аудитории без денег на подписки — стратегии 1–3 + механики контроля (голландский аукцион, «своя цена»).
D2C vs Маркетплейс

Иллюзия скидок и честная цена

Как устроена «скидка» на маркетплейсе
Цена «до» = 11 000 ₽ → Цена «сейчас» = 1 500 ₽ → Скидка -86%
Реальность
Товар никогда не продавался за 11 000 ₽. Это искусственно завышенная «зачёркнутая» цена для создания иллюзии выгоды. Покупатели уже научились не доверять таким скидкам.
Честная цена как стратегия D2C
1. Прозрачное ценообразование: показываете структуру цены (себестоимость + наценка) → доверие растёт

2. Скидка только как событие: «распродаём прошлую коллекцию» или «первая партия с браком маркировки» — людям верят, потому что это логично

3. «Справедливая цена»: диапазон 1 200–1 800 ₽, покупатель сам выбирает → психологически убивает маркетплейс

Игровые механики

Новизна, выбор, контроль

Психология маркетплейса
Причина: У людей нет денег на подписки, но есть потребность в новизне, выборе и контроле.
Следствие: Маркетплейс с бесконечным каталогом даёт иллюзию бесконечного выбора — но на самом деле там китайский ширпотреб. Покупатель устал от стресса выбора.
Как D2C даёт те же три потребности
Новизна → Дропы и микроколлекции: Новинки до выхода на маркетплейс, ограниченные партии, FOMO

Выбор → Кастомизация: Цвет, размер, гравировка — вы собираете свой товар. Это реальный выбор, а не выбор из 50 000 одинаковых

Контроль → Прозрачность: Видите реальную себестоимость, выбираете дату доставки, предлагаете идеи — бренд их воплощает

Игровые механики

Честная цена как оружие

Причинно-следственная цепочка доверия
Причина 1
Маркетплейсы массово применяют псевдоскидки
Товар за 1 500 ₽ помечается «было 11 000 ₽». Покупатель видит это постоянно.
Следствие 1
Цинизм и неверие в скидки
Покупатель больше не радуется скидке — он подозревает обман. «Скидка 90%» вызывает смех, а не желание купить.
Возможность D2C
Бренд с честной ценой = выигрыш на доверии
Простая цена без игр + объяснение (себестоимость + наценка) = мощное конкурентное преимущество. Для уставшего от лжи покупателя — глоток воздуха.
Следствие 2
Лояльность > скидки
Покупатель, который доверяет бренду, возвращается без рекламы. LTV растёт, CAC падает. Честность — дешевле скидок в долгосрочной перспективе.
